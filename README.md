## ğŸš€ Getting Started

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/Abdelrahman-Radaideh/portfoliotest.git
   cd portfoliotest
   ```

2. **Install dependencies**
   ```bash
   # Install frontend
   cd client && npm install

   # Install backend
   cd ../server && npm install
   ```

### Running the Application

1. **Backend Server**
   ```bash
   cd server
   npm run dev
   ```
   Server defaults to `http://localhost:3000`

2. **Frontend Client**
   ```bash
   cd client
   npm run dev
   ```
   Client defaults to `http://localhost:5173`
